## 安装方法
1.下载源码.
2.解压压缩包.
3.上传源码至服务器.
4.创建mysql数据库,并上传源码根目录的mysql.sql.
5.修改根目录下config.php的$conn数据库信息,指向第四步已经导入数据的mysql数据库.
6.修改根目录下config.php的$pass管理密码信息,防止被人滥用.
7.域名解析指向网站.
8.访问域名进行确认.
